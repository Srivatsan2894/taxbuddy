# TaxBuddy

India's Smart Salary & Tax Companion - Android App

## Features
- Calculate taxes for FY 2024-25, 2025-26, 2026-27
- Compare Old Regime vs New Regime
- Detailed tax breakdown
- Tax health score
- Educational resources

## Tech Stack
- React 18 + TypeScript
- Vite
- Capacitor (Android)

## Build & Deploy
```bash
npm install
npm run build
npm run cap:add:android
npm run cap:sync
cd android && ./gradlew assembleRelease
```
