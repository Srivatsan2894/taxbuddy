# TaxBuddy

India's Smart Salary & Tax Companion

A comprehensive Android app for calculating Indian income taxes and comparing tax regimes.

## Features
- Calculate taxes for FY 2024-25, 2025-26, 2026-27
- Compare Old Regime vs New Regime instantly
- Detailed tax breakdown with explanations
- Track deductions (80C, 80D, NPS, etc.)
- Tax health score
- Educational resources
- Dark mode support
- Offline functionality

## Build
```bash
npm install
npm run build
```

## Tech Stack
- React 18 + TypeScript
- Vite
- React Router
